Quando Usar Esta Ferramenta?

Esta funcionalidade é ideal para corrigir falhas em um arquivo HTML gerado pela conversão de PDF.

Se a sua conversão inicial resultou em páginas faltando ou com conteúdo incompleto, siga estes passos:

  1. Volte para a página de conversão de PDF.
  2. Envie o mesmo PDF novamente, mas no campo "Páginas", especifique apenas os números das páginas que apresentaram erro (ex: 4, 9-11).
  3. A conversão irá gerar um novo arquivo HTML menor, contendo somente as páginas reprocessadas.
  4. Use o formulário abaixo para unir os arquivos:
    • Arquivo HTML Original: Envie o arquivo grande gerado na primeira conversão (com as falhas).
    • Arquivo HTML com Correções: Envie o arquivo pequeno que você acabou de gerar.

O resultado será um arquivo HTML final, completo e corrigido, pronto para download.
